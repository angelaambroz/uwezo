Uwezo demo
=======

7 January 2015. A rough cut `d3.js` viz for the Uwezo folks to see what can be done.

### Main idea

Dynamic bar chart, by region, of kids' performance on English, Math, Swahili for 2012. L and R for years. Click for further information (side, informative tooltip?). Drop-down for countries (TZ, KE, UG).
